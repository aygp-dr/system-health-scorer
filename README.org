#+TITLE: system-health-scorer
#+AUTHOR: aygp-dr

Health score dashboard with drill-down. Terminal UI built with [[https://github.com/charmbracelet/bubbletea][Bubble Tea]].

** Run

#+begin_src bash
go run .
#+end_src

** Build

#+begin_src bash
make build
./bin/system-health-scorer
#+end_src

** Controls

- =j/k=: navigate
- =Enter=: select/details
- =q=: quit
