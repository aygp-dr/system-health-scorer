# system-health-scorer

Health score dashboard with drill-down. Terminal UI built with [Bubble Tea](https://github.com/charmbracelet/bubbletea).

## Run

```bash
go run .
```

## Build

```bash
make build
./bin/system-health-scorer
```

## Controls

- `j/k`: navigate
- `Enter`: select/details
- `q`: quit
